OpenDesk Agent v0.2.0

NOUVEAUTÉS
- Démarrage automatique avec Windows
- Exécution sous le compte SYSTEM
- Journal opendesk-agent.log
- Heartbeat automatique
- Synchronisation du carnet partagé
- Chemins fiables même au démarrage de Windows

INSTALLATION
1. Décompresser le pack dans un dossier permanent, par exemple :
   C:\Program Files\OpenDesk Agent

2. Modifier config.json si nécessaire.

3. Lancer build.bat.

4. Clic droit sur install_autostart.bat :
   Exécuter en tant qu'administrateur.

5. Vérifier dans OpenDesk Hub que le PC est en ligne.

FICHIERS CRÉÉS
- OpenDesk-Agent.exe
- opendesk-agent.log
- address_book.json

DÉSINSTALLATION
Lancer uninstall_autostart.bat en tant qu'administrateur.

IMPORTANT
Ne pas déplacer le dossier après l'installation du démarrage automatique.
